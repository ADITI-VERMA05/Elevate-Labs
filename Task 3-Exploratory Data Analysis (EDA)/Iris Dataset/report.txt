1. DISTRIBUTION OF NUMERICAL FEATURES (HISTOGRAMS)
-------------------------------------------------
Histograms of sepal length, sepal width, petal length, and petal width indicate that

features. Sepal features exhibit significant overlap among species, whereas petal
features show better separation, making them more useful for classification.

2. CATEGORICAL FEATURE ANALYSIS (COUNT PLOT)
--------------------------------------------
The count plot of the target variable (Species) shows that Setosa, Versicolor, and
Virginica classes are equally distributed. This confirms that the dataset is balanced
and does not suffer from class imbalance issues.

3. OUTLIER DETECTION (BOX PLOTS)
--------------------------------
Box plots reveal the presence of a few outliers, mainly in the sepal width feature.
Petal length and petal width contain fewer outliers and have compact distributions.
Overall, the dataset contains minimal noise and does not require extensive outlier
handling.

4. CORRELATION ANALYSIS (HEATMAP)
---------------------------------
The correlation heatmap shows a strong positive correlation between petal length and
petal width. Sepal length has moderate correlation with petal features, while sepal
width has weak correlation with other variables. This indicates that petal features are
highly related and significant for prediction.

5. KEY INSIGHTS FROM VISUALIZATIONS
----------------------------------
- Petal features provide better separation between iris species.
- Sepal features overlap significantly across classes.
- The dataset is balanced and clean.
- Correlation patterns highlight the importance of petal dimensions.
